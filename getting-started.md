# FakesomeFest Setup Guide
## For real Wii U's only.

Hi!
If you're here, that means you have a real Wii U and want to install FakesomeFest.

Requirements:
1. A copy of Splatoon.
2. A modded Wii U with Aroma. Follow [a guide](https://wiiu.hacks.guide) if you haven't hacked yours yet.
3. The [custom version](https://cdn.abmanagement.al/FakesomeFest.wps) of Pretendo's Inkay plugin.

If you have all of those, you're ready to go!

Continue to [Installing](/installing.html).
